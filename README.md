# Business Analytics MVP

Платформа бизнес-аналитики с AI-агентами на базе OpenRouter API. Предприниматели могут взаимодействовать со специализированными агентами для получения бизнес-инсайтов.

## Технологии

- **Frontend**: React 19 + Vite + TypeScript
- **Стилизация**: Tailwind CSS v4
- **LLM**: OpenRouter API (пользователь вводит свой ключ)
- **Парсинг файлов**: xlsx, mammoth, papaparse
- **Графики**: Recharts
- **State**: Zustand

## AI-агенты

| Агент | Описание |
|-------|----------|
| 📊 Финансовый аналитик | Загрузи документ с доходами и расходами — получи полный финансовый отчёт |
| 🔍 Анализ конкурентов | Введи сайты конкурентов — получи их слабые места и возможности |
| 🚚 Управление поставками | Загрузи список поставщиков и заказов — получи оптимизацию и риск-анализ |
| ⚖️ Юридический ассистент | Загрузи договор — получи анализ рисков и опасных пунктов |
| 📈 Прогноз продаж | Загрузи исторические данные продаж — получи прогноз и рекомендации |
| 👥 HR-аналитика | Загрузи данные по сотрудникам — получи анализ эффективности и ФОТ |

## Установка и запуск

### Требования

- Node.js 18+
- npm или yarn
- API-ключ от [OpenRouter](https://openrouter.ai)

### Шаги

```bash
# Клонировать репозиторий
git clone git@github.com:beckkyway/mvp-business-run.git
cd mvp-business-run

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

### Сборка для продакшена

```bash
npm run build
npm run preview
```

## Настройка API-ключа

1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai)
2. Получите API-ключ в разделе Keys
3. При первом запуске приложения введите ключ в модальном окне Settings
4. Ключ сохраняется в `localStorage` браузера

### Поддерживаемые модели

- `anthropic/claude-3-haiku` — быстрая и дешёвая (по умолчанию)
- `anthropic/claude-3.5-sonnet` — лучшее качество
- `openai/gpt-4o-mini` — альтернатива
- `google/gemini-flash-1.5` — самая дешёвая

## Поддерживаемые форматы файлов

- **PDF** — финансовые отчёты, договоры
- **Excel (.xlsx, .xls)** — данные продаж, HR, поставщики
- **CSV** — любые табличные данные
- **DOCX** — договоры, документы
